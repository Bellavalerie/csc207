HTML

HTML means HyperText Markup Language. HTML is a Marking-up Language for describing web documents or web pages.
The mark-up language is a set of markup tags and html documents are described by HTML tags.


Web document contains

  • The DocType declaration defines the document type to be HTML. The declaration help the browser to display web page
  • HTML Tags: It describe the html document
  • Head Tags: It provides information about the document.
  • Title Tag: Provides the title of the document
  • Body Tag: It describe the visible page content. Only body area are display by the browser.
  • Header: It describe the headings and it range from h1 to h6

HTML Tags: These are keywords or tag names surrounded by angle brackets, HTML tags normally comes in pair i.e the first tag and second tag. First (opening or start) Tag and Second (closing or end) Tag.
Browser: This are application used to read html documents and display them.
HTML Attribute: Attribute provide additional information about an element. Attribute are always specified in the start tag.
Font Tag need 3 attributes
  • Size: Default size of HTML is 3
    1. Representation of font size:
      Relative value
      Absolute value
  • Font: simply refers to the appearance. Font actually accept multiple value
  • Ordered list: is used when ordering is of importance. Its a container tag
  • Unordered list: it is the opposite of ordered list
  • Definition list: its used in defining terms.

Advantages Of HTML

  1. It is widely used
  2. Every browser supports HTML
  3. Easy to learn and use
  4. No need to purchase extra software i.e it is by default in every window
  5. You can integrate HTML with CSS and JAVASCRIPT

Disadvantages Of HTML

  1. Need to write a lot of code for making a simple web page
  2. Security features are not good in HTML
  3. It can create only static and plain pages
  4. It does not produce dynamic content

Web developing include two section
  • Back-end
  • Front-end (showed by client or users)

XML: Extensible mark-up language. it defines a set of rules for encoding documents in a format i.e both human-readable and machine-readable

Advantages Of XML

  1. XML is much simpler compare to SGML
  2. XML is very powerful, yet easy to implement
  3. It is in text format, Human readable
  4. Linking is very much simpler in XML than SGML but also much more intelligent than HTML
  5. Easy to decode and edit
  6. XML documents can be valid or well formatted without DTD

Difference between Client side and server side

Client is a system with browser and Internet connectivity while Server is a system or computer that provides services to other users.
